# ShopNest

ShopNest is a modern e-commerce web application built using React.

## Features

- Product Listing
- Product Details Page
- Search Products
- Category Filtering
- Price Sorting
- Dynamic Routing
- Loading State

## Technologies Used

- React
- React Router DOM
- CSS3
- DummyJSON API
